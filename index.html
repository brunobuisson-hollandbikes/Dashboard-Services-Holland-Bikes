1. Location Longue Durée (LLD)

1.1 GreenLeaze

GreenLeaze est notre prestataire pour la LLD (remplace MyPangee). Le contrat est entre le client et GreenLeaze. Plus rapide (souscription en ligne en quelques minutes) et vérification des fonds via Open Banking.

Argumentaires de vente

Pour un particulier : "Simple et rapide (~5 min), package tout compris (entretien, assistance, casse/vol), engagement 12/24/36 mois."

Argument financier : "Accessibilité (loyers mensuels vs achat comptant) et sérénité avec une offre clé en main."

Procédure de vente & Facturation (CycleSoftware)

1. Simulation : Saisir montant TTC (Vélo + Accessoires + Éco-part) + Entretien recommandé.
2. Dossier : Client reçoit le lien email/SMS (IBAN + signature électronique).
3. Livraison : Cliquer impérativement sur "Livrer le matériel" sur le dashboard pour déclencher le paiement.

Facturation CS : Établir au nom de Greenleaze SAS (RCS 984942573).
Inclure : N° Vélopass (mail: identification@greenleaze.com) + Ligne commission vendeur 2%.
Commission = Durée (mois) × 2 % × (loyer TTC / 1,2)

3. Assurance Laka

3.1 Assurance 4 mois offerte

Offerte pour tout achat de vélo. Couvre uniquement le VOL. Franchise de 10% (min 50€). Déclaration sous 48h sur laka.co.

Condition : Antivol agréé obligatoire + facture de l'antivol antérieure au sinistre.

3.2 Option 1 : Extension Prépayée (1 ou 2 ans)

À vendre directement lors de l'achat. Tarif : 7,5% de la valeur du vélo.

1 AN : Code 2181007168585
2 ANS : Code 2181007168622 (-20% remise)
Procédure CS : Ajouter l'article + Remarque avec dates. Ne pas cocher la case "assurance". 4 mois offerts démarrent le jour J, l'extension prend effet 4 mois plus tard.
Calculateur Assurance

3.3 Option 2 : Assurance mensuelle Laka

Si le client refuse l'extension : Cocher la case CS "assurer le vélo". Cela déclenche la suite d'emails Laka (J0, S1, M3, M3.25, M4). Pas de franchise en cas de vol.
